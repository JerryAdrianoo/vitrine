# Template — Aula

> Estrutura de uma aula técnica isolada. Para conteúdo dentro de um projeto, prefira `project-template.md`.

## 1. Objetivo
- O que o aluno saberá fazer ao final?
- Tier do roadmap que isso fecha?

## 2. Pré-requisitos
- Conceitos que o aluno precisa antes de começar (com link ou nota)
- Como verificar se ele tem (1 pergunta-âncora)

## 3. Motivação
- Que problema real esse conceito resolve?
- O que acontece sem ele? (cite incidente, anti-padrão ou trade-off)

## 4. Conceito
- Explicação direta — sem encheção
- Diagrama em texto se ajudar
- Princípios SOLID/Clean Code/Patterns aplicáveis (por extenso)

## 5. Código (linha por linha)
```java
// código com cada linha comentada com //
```
- Cada linha com `//` explicando o porquê
- Anti-padrões mencionados quando aplicável

## 6. Exercício
- Tarefa pequena para o aluno fazer sozinho
- Critério verificável de "pronto"
- O que entregar: arquivo X com método Y assinatura Z

## 7. Revisão
- O aluno entrega → leio o arquivo direto (não pedir para colar)
- Checklist da revisão (referenciar `../skills/teacher/references/teaching-principles.md` seção 5)
